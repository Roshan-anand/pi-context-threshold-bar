# pi-context-threshold-warn

extention for pi to show a warning bar at the bottom of chat when the context is about to reach the limit.
as most of the models starts degrading in performance even hellucinate when context window reach ~120k tokens, this extension will show a warning bar at the bottom of the chat when the context is about to reach the limit.

## Install

```bash
pi install git:github.com/Roshan-anand/pi-context-threshold-bar
```

Or from a local path:

```bash
pi install /path/to/pi-context-threshold-bar
```
